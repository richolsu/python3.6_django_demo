# Configure Mysql DB
```
'NAME': 'mysite',
'USER': 'root',
'PASSWORD': '',
'HOST': '127.0.0.1',
'PORT': '3306',
```

# Migrate Mysql DB
```
python manage.py makemigrations polls
python manage.py migrate
```

# Create Super User
```
python manage.py createsuperuser
```

# Run server
```
python manage.py runserver
```

# Access the web page.
```
http://127.0.0.1:8000/
```
